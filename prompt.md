A Python based MCP Client that specializes in testing and debugging MCP Servers. 

Here are some requirements:

- It should be Python based
- It should use UV
- It should use pydantic and pydantic_settings
- It should use FastMCP Client as it's MCP framework
- It should work directly in Python
- It should also work as a terminal app with a TUI
- The MCP Client should support passing a .env file in as an input arg
- The MCP Client should support these capabilities
  - Resources
  - Prompts
  - Tools
  - Discovery
  - Sampling
  - Roots
  - Elicitation
  - Instructions
